    Create `app/components/CelebrityCard.tsx`:

    Props:
    ```typescript
    interface CelebrityCardProps {
      card: CelebrityCard;
      index: number;       // for stagger delay
      onDismiss: () => void;
    }
    ```

    Wrap component in `React.memo` for performance (per PITFALLS.md — prevent re-render of entire list on each dismiss).

    **Framer Motion setup:**
    - `const x = useMotionValue(0);`
    - `const rotate = useTransform(x, [-200, 0, 200], [-15, 0, 15]);` (tilt +/-15deg)

    **Drag handler:**
    ```typescript
    function handleDragEnd(_: MouseEvent | TouchEvent | PointerEvent, info: PanInfo) {
      const shouldDismiss = Math.abs(info.offset.x) > 120 || Math.abs(info.velocity.x) > 500;
      if (shouldDismiss) {
        onDismiss();
      }
    }
    ```

    **motion.div props:**
    - `style={{ x, rotate, touchAction: "none" }}` — CRITICAL: touchAction inline prevents scroll conflict
    - `drag={card.isChad ? false : "x"}`
    - `dragElastic={0.15}`
    - `dragConstraints={{ left: 0, right: 0 }}`
    - `dragTransition={{ bounceStiffness: 400, bounceDamping: 40 }}`
    - `onDragEnd={card.isChad ? undefined : handleDragEnd}`
    - `initial={{ opacity: 0, y: 20 }}`
    - `animate={{ opacity: 1, y: 0, transition: { delay: index * 0.06, duration: 0.3, ease: "easeOut" } }}`
    - `exit={{ x: 600, opacity: 0, rotate: 30, transition: { duration: 0.25, ease: "easeIn" } }}`
    - `layout` prop for grid reflow animation

    **Card styling (Tailwind classes on the motion.div):**
    - `relative overflow-hidden rounded-lg`
    - Border: `border` with conditional class: `card.isChad ? "border-neon-yellow" : "border-neon-cyan/40 hover:border-neon-cyan"`
    - Background: `bg-cyber-panel`
    - Box shadow resting: `shadow-[0_0_8px_rgba(0,245,255,0.15)]`
    - Box shadow hover: `hover:shadow-[0_0_16px_rgba(0,245,255,0.35)]`
    - Aspect ratio: `aspect-[3/4]` on mobile, use responsive aspect ratio if needed
    - Transition for hover effects: `transition-shadow duration-200`

    **Card content layout (top to bottom):**
    1. Profile photo: `<img src={card.photoPath} alt={card.displayName} className="w-full h-[60%] object-cover" />` with `onError` fallback that hides image and shows initials
    2. Card body: `<div className="p-4 flex flex-col gap-1">`:
       - Display name: `<p className="text-base font-bold text-cyber-text">{card.displayName}</p>` (16px)
       - @handle: `<p className="text-xs font-bold text-neon-cyan">{card.handle}</p>` (12px)
       - Bio: `<p className="text-sm text-cyber-text line-clamp-2">{card.bio}</p>` (14px)
       - Follower count: `<p className="text-xs text-cyber-muted">{card.followerCount} followers</p>` (12px)

    **DismissButton:**
    - Render `<DismissButton onDismiss={onDismiss} />` only when `!card.isChad`

    **Image error fallback:**
    - On `<img onError>`, hide the image and show a `<div>` with the celebrity's initials (first letter of first and last name) in cyber-muted color on a cyber-surface background.
